# Weekly Number Bingo Club

Netlify-ready React + Vite prototype.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to Netlify

1. Upload this folder to a GitHub repo.
2. In Netlify, click **Add new project**.
3. Connect your GitHub repo.
4. Build command: `npm run build`
5. Publish directory: `dist`
6. Deploy.

## Important

This version uses `localStorage`, so it is only a single-browser prototype.
For real multi-user play, add a backend like Supabase or Firebase.
